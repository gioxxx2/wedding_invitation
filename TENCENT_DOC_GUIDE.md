## 功能说明

### 1. 自动保存
- ✅ 所有来宾提交的信息都会自动保存到浏览器本地存储
- ✅ 数据不会丢失，即使刷新页面

### 2. 同步到腾讯文档

#### 方法一：一键同步（推荐）

1. 在请柬页面底部，点击 **"📋 同步到表格"** 按钮
2. 数据会自动复制到剪贴板
3. 腾讯文档会自动打开
4. 在表格中选中要粘贴的起始单元格（建议从第2行开始，第1行是表头）
5. 按 **Ctrl+V** (Windows) 或 **Cmd+V** (Mac) 粘贴数据

**数据格式：**
```
姓名 | 电话 | 参加人数 | 祝福语 | 提交时间
```

#### 方法二：导出CSV文件

1. 点击 **"📥 导出数据"** 按钮
2. 下载 CSV 文件
3. 在腾讯文档中，点击 **"文件" → "导入" → "上传文件"**
4. 选择下载的 CSV 文件即可导入

### 3. 查看表格

点击 **"📊 查看来宾信息表格"** 按钮，可以直接打开腾讯文档查看所有数据。

## 腾讯文档表格设置建议

### 表头设置

建议在腾讯文档表格的第一行设置以下表头：

| 姓名 | 电话 | 参加人数 | 祝福语 | 提交时间 |
|------|------|----------|--------|----------|

### 数据格式

- **姓名**：文本
- **电话**：文本
- **参加人数**：数字（1-5及以上）
- **祝福语**：文本（可能为空）
- **提交时间**：日期时间格式

## 使用流程

### 对于来宾：
1. 填写表单并提交
2. 信息自动保存

### 对于管理员：
1. **定期同步数据：**
   - 点击 "📋 同步到表格" 按钮
   - 在腾讯文档中粘贴数据
   
2. **或导出备份：**
   - 点击 "📥 导出数据" 按钮
   - 保存 CSV 文件作为备份

3. **查看统计：**
   - 点击 "📊 查看来宾信息表格" 查看实时数据
   - 在腾讯文档中可以使用筛选、排序等功能

## 高级功能（可选）

### 设置自定义表格链接

如果更换了腾讯文档表格：

1. 点击 **"⚙️ 设置表格链接"** 按钮
2. 输入新的腾讯文档链接
3. 保存即可

### 使用 Webhook 自动同步（需要配置）

如果需要实现完全自动同步，可以：

1. 使用腾讯轻联等第三方服务
2. 配置 Webhook URL
3. 数据会自动同步到腾讯文档

## 注意事项

1. **数据隐私**：所有数据首先保存在浏览器本地，只有管理员可以同步到腾讯文档
2. **数据备份**：建议定期导出 CSV 文件作为备份
3. **表格权限**：确保腾讯文档表格设置为"任何人可编辑"或"链接分享"
4. **数据格式**：粘贴数据时，确保从正确的单元格开始，避免覆盖表头

## 常见问题

**Q: 数据没有自动同步到腾讯文档？**
A: 由于腾讯文档没有公开API，需要手动点击"同步到表格"按钮。数据已保存在本地，不会丢失。

**Q: 可以设置自动同步吗？**
A: 可以使用腾讯轻联等第三方服务配置 Webhook，实现自动同步。

**Q: 如何批量导入数据？**
A: 使用"导出数据"功能下载CSV文件，然后在腾讯文档中导入。

**Q: 数据会丢失吗？**
A: 不会。所有数据都保存在浏览器本地存储中，即使关闭浏览器也不会丢失。

## 技术支持

如有问题，请检查：
1. 浏览器是否支持剪贴板API
2. 腾讯文档链接是否正确
3. 浏览器控制台是否有错误信息
