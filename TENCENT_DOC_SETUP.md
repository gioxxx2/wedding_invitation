# 腾讯文档在线表格设置说明

## 方法一：手动创建表格并设置链接（推荐）

1. **创建腾讯文档表格**
   - 访问 [腾讯文档](https://docs.qq.com/)
   - 点击"新建" → "在线表格"
   - 创建表格，设置表头：`姓名`、`电话`、`参加人数`、`祝福语`、`提交时间`

2. **获取表格链接**
   - 在表格页面，点击右上角"分享"
   - 选择"链接分享"，设置为"任何人可查看"
   - 复制链接

3. **在请柬页面设置链接**
   - 在来宾信息收集页面，点击"设置表格链接"按钮
   - 粘贴刚才复制的腾讯文档链接
   - 设置完成后，来宾可以点击"查看来宾信息表格"查看所有信息

4. **手动导入数据**
   - 在请柬页面点击"导出数据"按钮，下载CSV文件
   - 在腾讯文档表格中，点击"文件" → "导入" → "上传文件"
   - 选择下载的CSV文件即可导入

## 方法二：使用腾讯文档API（高级）

如果需要自动同步数据到腾讯文档，需要：

1. **申请腾讯文档API**
   - 访问腾讯云控制台
   - 申请腾讯文档API权限
   - 获取API密钥

2. **配置Webhook**
   - 在 `script.js` 中的 `syncToTencentDoc` 函数中配置API
   - 或者使用腾讯文档的Webhook功能

## 方法三：使用其他在线表格服务

也可以使用：
- Google Sheets（需要科学上网）
- 飞书多维表格
- 石墨文档
- 其他支持导入CSV的在线表格服务

## 注意事项

- 所有来宾信息都会保存在浏览器本地存储中
- 即使没有设置腾讯文档链接，数据也不会丢失
- 可以随时导出CSV文件进行备份
- 建议定期导出数据作为备份
